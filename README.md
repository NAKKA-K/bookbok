# BOOKBOK

# Install
composer & npm install

```bash
$ make install
```

# Run
Run laravel server.

```bash
$ make run
```

# Help
View the all command list.

```bash
$ make help
```

# Build react
Run webpack.

```bash
$ make build
```

# Debug on laravel
Run debugger on laravel tinker.

```bash
$ make c
```
